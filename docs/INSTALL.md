# Installation

## npm (recommended)

```bash
npm install -g @talocode/contextlane
contextlane demo
```

## npx

```bash
npx @talocode/contextlane demo
```

## Linux / macOS

```bash
curl -fsSL https://raw.githubusercontent.com/talocode/contextlane/main/install.sh | bash
```

## Windows

```powershell
iwr -useb https://raw.githubusercontent.com/talocode/contextlane/main/install.ps1 | iex
```

## Android (Termux)

```bash
pkg install nodejs
npm install -g @talocode/contextlane
contextlane demo
```

## iOS

iOS support is available through iSH shell or by using the remote API.
No native iOS app ships in v0.1.0.
